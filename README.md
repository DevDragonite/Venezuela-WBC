## English

### Purpose
**WBC 2026 Manager - Venezuela Edition** is a strategic management and lineup visualization tool designed by fans, for fans of Venezuelan baseball. The goal is to allow users to simulate the roster selection, batting order, and starting rotation for the Venezuelan team in the 2026 World Baseball Classic.

### Key Features
- **Lineup Management**: Tactical selection and ordering of the 9 starters.
- **Starting Rotation**: Defining the pitchers for key group stage games.
- **Real-Time Visualization**: Dynamic defensive diamond reflecting the players' actual field positions.
- **Premium Export**: Generation of a shareable card (1080x1080) optimized for social media (Instagram, Twitter, WhatsApp) with Official Manager attribution.

### Tech Stack
- **Frontend**: [React](https://reactjs.org/) with [Vite](https://vitejs.dev/) for ultra-fast development.
- **Styling**: [Tailwind CSS](https://tailwindcss.com/) for a premium, responsive aesthetic.
- **Global State**: [Zustand](https://github.com/pmndrs/zustand) for lightweight and efficient state management.
- **Media Export**: [html-to-image](https://github.com/tsayen/html-to-image) to convert DOM components into high-resolution PNG images.
- **Icons**: [Lucide React](https://lucide.dev/).
